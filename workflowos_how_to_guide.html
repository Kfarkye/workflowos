Frequently Asked Questions
ü§î What's the difference between Link Manager and Dashboard?
Link Manager is your smart bookmark hub - it organizes all your tools and websites into clickable tiles.

Dashboard is your complete digital workspace that includes Link Manager plus document storage, AI tools, workflow templates, and automation.

Think of Link Manager as your phone's home screen, and Dashboard as your entire phone's operating system.
üè• Do I need Nova OS if I'm not in healthcare?
No! Nova OS is specifically designed for healthcare recruiting workflows. If you're in a different field, focus on Core Tools and FlowOS. The system is designed to be useful for any profession.
üíæ How do I backup my WorkflowOS data?
Enable Founder Mode (Cmd+Shift+D) and use the "Export DNA" feature to download your complete system configuration. You can also export individual tool data from their respective settings menus.
üîÑ Can I sync WorkflowOS across multiple devices?
Currently, WorkflowOS runs locally in your browser. Use the export/import features to transfer your configuration between devices. Cloud sync is planned for future releases.
üéØ What's the best workflow for beginners?
Start with Link Manager ‚Üí add your daily tools ‚Üí try Canvas for note-taking ‚Üí explore Prompt Vault for AI templates. Most users find their rhythm within the first week of daily use.
